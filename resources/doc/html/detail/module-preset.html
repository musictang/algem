Sélection de formules

Sélectionnez les formules qui vous intéressent. La sélection peut être conjointe (Maj) ou disjointe (Ctrl).

Présélections

  • Cliquez sur Ajouter pour mémoriser votre sélection.
  • Cliquez sur une ligne dans la liste de gauche pour rappeler la sélection.
  • Cliquez droit sur une présélection pour la renommer.
  • Cliquez sur Supprimer pour effacer une présélection.
