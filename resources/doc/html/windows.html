Menu Fenêtres

Détacher
Cette action est pour le moment inopérante.
Fermer
Ferme la fenêtre active.
Fermer toutes
Ferme toutes les fenêtres actuellement ouvertes, excepté la fenêtre de postit.
Iconifier toutes
Iconifie toutes les fenêtres actuellement ouvertes, excepté la fenêtre de postit.
Ouvrir toutes
Ouvre toutes les fenêtres actuellement à l'état d'icone.
Cascade
Place les fenêtres les unes au-dessus des autres, en cascade.

Les fenêtres actives sont visibles en bas du menu, après le trait de séparation.
