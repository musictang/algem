Menu Catalogue

Formule
Editer et créer des formules (ensemble de cours prédéfinis).
Cours
Editer et créer des cours.
Inscriptions
Afficher les inscriptions sur la période. Vous pouvez également supprimer une inscription.
Commandes formules
Afficher la liste des formules (ou programmes de formation) sélectionnées durant une certaine période. Par défaut, la période de recherche correspond á l'année scolaire.
Scripts
Ce menu n'est visible que si l'option « feature.scripting»  est activée dans les paramètres de démarrage. Ces scripts, s'ils existent, vous permettent d'effectuer des requêtes en base de données, sans avoir besoin de remanier le logiciel.
Importer photos
Importer l'ensemble des photos d'un répertoire. Seuls sont pris en compte les fichiers de type « image » contenus dans le dossier sélectionné. Les fichiers importés remplacent ceux de même nom en base de données.
Exporter photos
Exporter les photos vers un répertoire de votre choix. Les photos présentes dans le dossier de destination seront remplacées par celles portant le même nom.
